# 🔥 Z3R0-T3RM - Root Terminal Emulator (Kali Style)

**La terminal definitiva para Linux y Windows con poder ROOT/ZSH**

## Características
✅ Full root access (Linux `sudo su` / Windows ` administrator`)  
✅ ZSH + Powerlevel10k (tema Kali-like)  
✅ Aliases ofensivos y pentesting  
✅ Scripts de privilegios y persistencia  
✅ Compatible: Kali, Ubuntu, Parrot, WSL2, Windows Terminal
